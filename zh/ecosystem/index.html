TensorPlay

TensorPlay 是一个开源深度学习框架，专为教育和研究而设计，具有灵活和模块化的特点，提供了 AI 系统内部机制的清晰视图。

TensorPlay 简化了从理论理解到实践实现的路径。TensorPlay 旨在提供最大的清晰度和可破解性，支持动态计算图，使学生和研究人员能够直观地探索神经网络内部。其 Pythonic 设计以及与原生 Python 工具的深度集成，使其成为掌握深度学习概念的易用平台。

TensorPlay 专为教育和透明度而设计，是理解框架底层工作原理的理想游乐场。通过模块化的库生态系统，它支持从基础张量操作到构建自定义神经网络层的基本用例。TensorPlay 提供了 CPU 和 CUDA 后端的可读实现，非常适合学习如何编写自定义硬件内核并使用 nanobind 进行绑定。

无论你是正在学习深度学习基础知识，还是在通过原型验证新的架构想法，TensorPlay 都能提供将好奇心转化为知识所需的简洁性、透明度和灵活性。
